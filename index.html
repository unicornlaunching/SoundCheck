Zounds Javascript / Typescript Audio Library

Zounds is a javascript / typescript Audio Library that can add simple responsive sounds to your website or provide an audio engine for an entire javascript game.

Zounds works with all browsers that support the Web Audio API whist encapsulating that API's asynchronous nature.

The non-minified version has a full set of JSDoc comments.

What can it do?

  • Add simple noises to simple pages.
  • ...or add rich audio to larger single page applications like furl / unfurl noises on your menus, or add outcome relevant noises to popups.
  • Play multiple sounds at once.
  • Play mp3, ogg, or pretty much any other audio mime type the internet cares about.
  • Invisibly handle caching on a forgiving and just-in-time basis but also allow you to manually pre-cache as required.
  • ... and register a call back on the outcome of the combination of sounds precached.
  • Add anoying elevator music to your site and then stop it (if you're feeling charitable).
  • Make sure that as much of the internet goes boing as possible.
  • Permissive (MIT) open source licence
